# SaleXpert

**Live site:** [https://salexpert.co](https://salexpert.co)

Marketing site for SaleXpert — professional websites for local service businesses (plumbers, electricians, dentists, and similar). Built with [Next.js](https://nextjs.org/).

## Environment

Copy `.env.example` to `.env` locally. For production, set the same variables in your host (e.g. Vercel), including `NEXT_PUBLIC_SITE_URL=https://salexpert.co`.
